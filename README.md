This repository contains implementations of physics-informed neural
networks (PINNs) using JAX (with Equinox and Optax).

## Implemented Methods

### PINNs
- Static bar (1D)
- Static plate (2D)
- Heat conduction (1D, continuous-time)

### VPINNs (Variational PINNs)
- Burger's equation (1D)
- Thermoelastic damping (TED) in a vibrating ring

### Deep Ritz 
- 2D static plate
